# Lautech Studio

Site institucional da Lautech Studio, preparado para publicação no GitHub Pages.

## Estrutura

- `index.html`: página principal.
- `assets/styles.css`: estilos responsivos.
- `assets/script.js`: comportamento simples da página.
- `assets/brand-board.png`: imagem de identidade visual.
- `assets/homepage-concept.png`: conceito visual usado no destaque da página.
- `.github/workflows/pages.yml`: workflow para publicar no GitHub Pages usando Actions.

## Publicação no GitHub Pages

No GitHub, acesse `Settings > Pages` e selecione `GitHub Actions` como fonte de publicação.
Depois disso, cada push na branch `main` publicará uma nova versão.
